# Ridge 回帰係数による重要度

## 概要

Ridge 回帰（L2 正則化線形回帰）は、パラメータと目的関数の間に**線形関係**を仮定してモデルを適合させ、その係数 β を重要度として使う手法。係数の絶対値 `|β|` が大きいほど、そのパラメータが目的関数に強く寄与していることを示す。

---

## 数式

### モデル

```
ŷ = β_0 + β_1 x_1 + β_2 x_2 + ... + β_p x_p
```

Ridge 回帰は、通常の最小二乗法に L2 ペナルティを加えた以下の最適化問題を解く:

```
minimize: ‖y - Xβ‖² + α ‖β‖²
```

ここで α は正則化強度（実装では α = 1.0 固定）。

### 正規方程式

閉形式解:

```
β = (X^T X + αI)^{-1} X^T y
```

---

## 実装の詳細

### 前処理: Z スコア標準化

係数を比較可能にするため、X の各列を標準化してから回帰する:

```
x̃_j = (x_j - μ_j) / σ_j
```

σ_j ≈ 0 の場合は σ_j = 1.0 で除算（定数列のゼロ除算を回避）。  
y は平均センタリングのみ（y_c = y - ȳ）。

### 連立方程式の求解: ガウス消去法

`gaussian_elimination()` 関数（`sensitivity.rs:191`）が部分ピボット選択付きガウス消去法で (X^T X + αI) β = X^T y を解く。

- 計算量: O(p³)（パラメータ数 p が支配）
- p = 0 または行列が特異な場合は β = 0 を返す

### `compute_ridge()` の流れ（`sensitivity.rs:249`）

```
1. transpose_and_standardize(X)      → 列並び替え + Z スコア標準化
2. X^T X を計算（対称行列）          → O(n p²)
3. 対角に α を加算（Ridge 正則化）
4. X^T y を計算
5. gaussian_elimination で β を求解
6. R² を計算: 1 - SS_res / SS_tot
```

### 複数目的関数への対応

各目的関数に対して個別に Ridge 回帰を実行し、パラメータ p_j のスコアは全目的関数の |β| 平均:

```
score(p_j) = (1/m) Σ_k |β_jk|
```

---

## R² について

返り値の `r_squared` は Ridge モデルの決定係数:

```
R² = 1 - Σ(y_i - ŷ_i)² / Σ(y_i - ȳ)²
```

R² が低い（線形モデルが当てはまらない）場合、β の絶対値は重要度指標として信頼性が下がる。その場合は Spearman や Sobol の使用を推奨。

---

## 特性・限界

**強み:**

- 係数の符号から「増やすと目的関数が増える/減る」方向性がわかる
- 多重共線性があっても Ridge 正則化で安定化される
- 計算が非常に高速: O(n p²)

**弱み:**

- 線形関係を仮定しているため、非線形な影響を過小評価する場合がある
- 交互作用項を含まないため、パラメータ間の相乗効果は捉えられない
- 係数の大きさはスケールに依存する（標準化により緩和されているが完全ではない）

---

## 使用場面の目安

- 目的関数が概ね線形と期待されるとき
- 重要なパラメータを素早く特定してトライアルを絞り込みたいとき
- 係数の方向性（正負）も知りたいとき（ImportanceChart では |β| のみ表示だが、`sensitivityResult.ridge[k].beta[pi]` で取得可能）
