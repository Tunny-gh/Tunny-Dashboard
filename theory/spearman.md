# Spearman 順位相関係数

## 概要

スピアマン順位相関係数 ρ は、パラメータ x_i と目的関数 y の間の**単調な関係の強さ**を測るノンパラメトリック統計量である。値が線形関係に従わなくても、順序（大小関係）が保たれていれば高い値を示す。

ImportanceChart では `|ρ|`（絶対値）を表示する。正負の方向は問わず、「どれだけ強く関係しているか」のみを評価する。

---

## 数式

### Step 1: 順位変換

n 個のトライアルに対し、各変数の値を順位に変換する。

```
x = [3.1, 1.2, 4.5, 2.0]
rank(x) = [3, 1, 4, 2]
```

同値タイの処理: 同じ値が複数ある場合、それらの順位の平均を割り当てる（平均順位法）。

```
x = [1.0, 2.0, 2.0, 3.0]  →  rank(x) = [1, 2.5, 2.5, 4]
```

### Step 2: ピアソン相関を順位に適用

順位変換後の R_x, R_y に対してピアソン積率相関係数を計算する。

```
ρ = Corr(R_x, R_y) = Σ (R_xi - R̄_x)(R_yi - R̄_y) / √[Σ(R_xi - R̄_x)² · Σ(R_yi - R̄_y)²]
```

タイがない場合は以下の簡易式と等価:

```
ρ = 1 - 6 Σd_i² / (n(n² - 1))
```

ここで d_i = R_xi - R_yi（順位の差）。

## 実装の詳細

### `rank()` 関数（`sensitivity.rs:47`）

- NaN は常に末尾に配置（`Ordering::Greater`）
- 同値グループを特定し、グループ内の平均順位を割り当てる
- O(n log n) の計算量（ソート支配）

### `compute_spearman()` 関数（`sensitivity.rs:135`）

```rust
pub fn compute_spearman(x: &[f64], y: &[f64]) -> f64 {
    let rx = rank(x);
    let ry = rank(y);
    pearson_correlation(&rx, &ry)  // 順位列に対してピアソン相関
}
```

### 複数目的関数への対応

パラメータ p_j の重要度スコアは、全目的関数に対する |ρ| の平均:

```
score(p_j) = (1/m) Σ_k |ρ(p_j, y_k)|
```

ImportanceChart.tsx でこの平均化を実施している（`/ nObj`）。

## 特性・限界

**強み:**

- 外れ値に頑健（値そのものでなく順位を使う）
- 非線形でも単調な関係なら捉えられる
- 前提分布なし（ノンパラメトリック）
- 計算が高速: O(n log n)

**弱み:**

- 非単調な関係（U 字形など）は検出できない
- パラメータ同士の**交互作用**（x_1 と x_2 が組み合わさった効果）は捉えられない
- スケールを持たない（どの程度影響するかの量的評価が難しい）

---

## 使用場面の目安

- トライアル数が少ない（n < 50）場合でも安定して動作する
- まず全パラメータをざっくりスクリーニングしたいとき
- 目的関数の形状が未知のとき
