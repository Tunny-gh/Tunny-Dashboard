### 4. score 計算の分岐追加 🔵

**信頼性**: 🔵 *dataflow.md 機能3 ImportanceChart 表示フローより*

**実装ファイル**: `frontend/src/components/charts/ImportanceChart.tsx`

```typescript
// sensitivityResult.paramNames.map 内
const importances = sensitivityResult.paramNames.map((name: string, pi: number) => {
  let score = 0
  if (metric === 'spearman') {
    // 既存実装
  } else if (metric === 'beta') {
    // 既存実装
  } else if (metric === 'sobol_first' && sobolResult) {
    const vals = sobolResult.firstOrder[pi]
    score = vals.reduce((sum: number, v: number) => sum + v, 0) / vals.length
  } else if (metric === 'sobol_total' && sobolResult) {
    const vals = sobolResult.totalEffect[pi]
    score = vals.reduce((sum: number, v: number) => sum + v, 0) / vals.length
  }
  return { name, score }
})
```

**注意**: `sobolResult` が null のときは score = 0 のまま（`sobolResult &&` ガードで安全）
