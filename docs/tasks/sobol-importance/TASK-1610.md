### 3. build_quad_features() 🔵

**信頼性**: 🔵 *architecture.md 特徴量構成テーブルより*

p 個の標準化済み入力から二次特徴量ベクトルを構築する。
特徴量構成（合計 p(p+3)/2 個）:
- 線形項: x_0, ..., x_{p-1}  → p 個
- 二乗項: x_0², ..., x_{p-1}²  → p 個
- 交差項: x_0·x_1, x_0·x_2, ..., x_{p-2}·x_{p-1}  → p(p-1)/2 個

```rust
fn build_quad_features(x_std: &[f64]) -> Vec<f64> {
    let p = x_std.len();
    let n_feats = p + p + p * (p - 1) / 2;
    let mut feats = Vec::with_capacity(n_feats);
    // 線形項
    for &v in x_std { feats.push(v); }
    // 二乗項
    for &v in x_std { feats.push(v * v); }
    // 交差項
    for i in 0..p {
        for j in (i + 1)..p {
            feats.push(x_std[i] * x_std[j]);
        }
    }
    feats
}
```

### 4. lcg_next() 🔵

**信頼性**: 🔵 *architecture.md LCG64 PRNG より（外部クレート不使用）*

```rust
fn lcg_next(state: &mut u64) -> f64 {
    *state = state.wrapping_mul(6364136223846793005)
                  .wrapping_add(1442695040888963407);
    ((*state >> 11) as f64) / (1u64 << 53) as f64
}
```

### 5. build_sobol_surrogate() 🔵

**信頼性**: 🔵 *architecture.md Step 1・implementation-guide.md より*

既存の `DataFrame` からパラメータ行列 X とターゲット Y を取得し、Z スコア標準化後に二次特徴量を構築して Ridge 回帰（α=1.0）を適合させる。

実装の概要:
1. DataFrame から `param_names`・`objective_names` を取得
2. n < 2 または p == 0 の場合 `None` を返す
3. 各パラメータを Z スコア標準化し `(param_means, param_stds)` を保存
4. `build_quad_features()` で二次特徴量を構築
5. 二次特徴量を再標準化し `(quad_feat_means, quad_feat_stds)` を保存
6. 各目的関数に対して Ridge（λ=1.0）を適合し `(betas, intercepts)` を取得

### 6. compute_sobol() 🔵

**信頼性**: 🔵 *architecture.md Step 2-4・Jansen 推定量より*

```rust
pub fn compute_sobol(n_samples: usize) -> Option<SobolResult> {
    // 1. サロゲート構築
    let surrogate = build_sobol_surrogate()?;
    let p = surrogate.n_params;
    let n_obj = surrogate.betas.len();

    // 2. Saltelli 行列生成（LCG64）
    //    A: n_samples × p、B: n_samples × p

    // 3. AB_i 行列（A の第 i 列を B の第 i 列で置換）

    // 4. 各行列の全行でサロゲートを評価

    // 5. Jansen 推定量
    //    Var_Y  = mean(f_A²) - mean(f_A)²
    //    S_i    = mean(f_B * (f_AB_i - f_A)) / Var_Y
    //    ST_i   = mean((f_A - f_AB_i)²) / (2 * Var_Y)
    //    結果を [0, 1] にクリップ

    Some(SobolResult { param_names, objective_names, first_order, total_effect, n_samples })
}
```
