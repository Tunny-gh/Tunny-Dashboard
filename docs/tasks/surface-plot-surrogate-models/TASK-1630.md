### 4. 予測関数 🔵

**信頼性**: 🔵 *architecture.md rf.rs 設計より*

```rust
pub(crate) fn predict_one(node: &TreeNode, x: &[f64]) -> f64 {
    match node {
        TreeNode::Leaf(v) => *v,
        TreeNode::Split { feature, threshold, left, right } => {
            if x[*feature] <= *threshold {
                predict_one(left, x)
            } else {
                predict_one(right, x)
            }
        }
    }
}
```

## 単体テスト要件

### テストケース1: 完全分離可能なデータで正しく分岐 🔵

**信頼性**: 🔵 *architecture.md CART アルゴリズム設計より*

**Given**: x = [[0.0], [1.0], [2.0], [3.0]], y = [0.0, 0.0, 1.0, 1.0]（x >= 2.0 で y=1）
**When**: `build_tree()` で深さ 1 の木を構築し `predict_one()` で各点を予測
**Then**: x=0, x=1 で 0.0、x=2, x=3 で 1.0 を返す

**テストファイル**: `rust_core/src/rf.rs`（`#[cfg(test)]` モジュール）
