## 実装詳細

### 1. computeSurface3d のバグ修正 🔵

**信頼性**: 🔵 *architecture.md「analysisStore.ts の既存バグ修正（surrogateModelType が WASM に渡っていない）」より*

**実装ファイル**: `frontend/src/stores/analysisStore.ts`（現在の `analysisStore.ts:104`）

```typescript
// 変更前（analysisStore.ts:104）
const result = wasm.computePdp2d(param1, param2, objective, nGrid)

// 変更後
const result = wasm.computePdp2d(param1, param2, objective, nGrid, surrogateModelType)
```

`surrogateModelType` は同関数の先頭で `get()` から取得済み（`analysisStore.ts:95`）:

```typescript
const { surrogateModelType, surface3dCache } = get()
```

変更はこの 1 行のみ。
