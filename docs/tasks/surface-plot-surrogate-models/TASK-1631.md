### 1. シンプルな疑似乱数生成器 🟡

**信頼性**: 🟡 *純 Rust 実装制約から妥当な推測（外部クレート禁止）*

**実装ファイル**: `rust_core/src/rf.rs`

```rust
/// 線形合同法による疑似乱数生成器
struct Lcg {
    state: u64,
}

impl Lcg {
    fn new(seed: u64) -> Self { Self { state: seed } }
    fn next_u64(&mut self) -> u64 {
        self.state = self.state.wrapping_mul(6364136223846793005)
            .wrapping_add(1442695040888963407);
        self.state
    }
    fn next_usize(&mut self, n: usize) -> usize {
        (self.next_u64() % n as u64) as usize
    }
}
```

### 2. Bootstrap サンプリング 🔵

**信頼性**: 🔵 *design-interview.md Q5（Bootstrap + Bagging）より*

```rust
pub(crate) fn bootstrap_sample(
    x: &[Vec<f64>],
    y: &[f64],
    rng: &mut Lcg,
) -> (Vec<Vec<f64>>, Vec<f64>) {
    let n = y.len();
    let indices: Vec<usize> = (0..n).map(|_| rng.next_usize(n)).collect();
    let x_boot = indices.iter().map(|&i| x[i].clone()).collect();
    let y_boot = indices.iter().map(|&i| y[i]).collect();
    (x_boot, y_boot)
}
```

### 3. RandomForest 構造体と学習 🔵

**信頼性**: 🔵 *architecture.md RandomForest 構造体・デフォルトパラメータより*

```rust
pub(crate) struct RandomForest {
    pub trees: Vec<DecisionTree>,
}

pub(crate) fn train_random_forest(
    x: &[Vec<f64>],
    y: &[f64],
    n_trees: usize,    // デフォルト: 100
    max_depth: usize,  // デフォルト: 10
    min_samples_leaf: usize, // デフォルト: 2
    seed: u64,
) -> RandomForest {
    let mut rng = Lcg::new(seed);
    let n_features = if x.is_empty() { 0 } else { x[0].len() };
    let feature_indices: Vec<usize> = (0..n_features).collect();
    let trees = (0..n_trees).map(|_| {
        let (x_boot, y_boot) = bootstrap_sample(x, y, &mut rng);
        let root = build_tree(&x_boot, &y_boot, &feature_indices, 0, max_depth, min_samples_leaf);
        DecisionTree { root }
    }).collect();
    RandomForest { trees }
}
```

### 4. アンサンブル予測 🔵

**信頼性**: 🔵 *architecture.md RF 予測設計より*

```rust
pub(crate) fn rf_predict(forest: &RandomForest, x: &[f64]) -> f64 {
    let n = forest.trees.len();
    if n == 0 { return 0.0; }
    forest.trees.iter().map(|t| predict_one(&t.root, x)).sum::<f64>() / n as f64
}

pub(crate) fn rf_r_squared(forest: &RandomForest, x: &[Vec<f64>], y: &[f64]) -> f64 {
    let y_mean = y.iter().sum::<f64>() / y.len() as f64;
    let ss_tot: f64 = y.iter().map(|&v| (v - y_mean).powi(2)).sum();
    let ss_res: f64 = x.iter().zip(y).map(|(xi, &yi)| {
        let pred = rf_predict(forest, xi);
        (yi - pred).powi(2)
    }).sum();
    if ss_tot < f64::EPSILON { return 1.0; }
    1.0 - ss_res / ss_tot
}
```
