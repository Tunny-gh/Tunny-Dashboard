## 実装詳細

### 1. alpha = K^{-1}y 計算 🔵

**信頼性**: 🔵 *architecture.md「alpha = L^T \ (L \ y)」より*

**実装ファイル**: `rust_core/src/kriging.rs`

```rust
/// alpha = K^{-1} y を Cholesky 因子 L を使って計算
pub(crate) fn compute_alpha(l: &[Vec<f64>], y: &[f64]) -> Vec<f64> {
    let v = forward_sub(l, y);   // v = L^{-1} y
    backward_sub(l, &v)           // alpha = L^{-T} v = K^{-1} y
}
```

### 2. 対数周辺尤度 🔵

**信頼性**: 🔵 *architecture.md「L = −½ y^T α − Σ log(L_ii) − n/2 log(2π)」より*

```rust
pub(crate) fn log_marginal_likelihood(
    x: &[Vec<f64>],
    y: &[f64],
    log_ls: &[f64],
    log_sf: f64,
    log_sn: f64,
) -> f64 {
    let k = build_kernel_matrix(x, log_ls, log_sf, log_sn);
    let l = match cholesky(&k) {
        Some(l) => l,
        None => return f64::NEG_INFINITY,
    };
    let alpha = compute_alpha(&l, y);

    // -½ y^T α
    let data_fit: f64 = y.iter().zip(&alpha).map(|(yi, ai)| yi * ai).sum();
    // -Σ log(L_ii)  (log行列式の半分)
    let log_det: f64 = l.iter().enumerate().map(|(i, row)| row[i].ln()).sum();
    // -n/2 log(2π)
    let n = y.len() as f64;

    -0.5 * data_fit - log_det - 0.5 * n * (2.0 * std::f64::consts::PI).ln()
}
```

### 3. Matérn 5/2 の偏微分（解析的勾配） 🔵

**信頼性**: 🔵 *architecture.md「∂k/∂log(l_d) = σ_f² · 5/3 · (x1_d−x2_d)²/l_d² · (1+√5·r) · exp(−√5·r)」より*

```rust
/// Matérn 5/2 の ∂k/∂log(l_d) （ARD の次元 d に対する偏微分）
fn matern52_ard_grad_ld(x1: &[f64], x2: &[f64], log_ls: &[f64], log_sf: f64, dim: usize) -> f64 {
    let sigma_f2 = (2.0 * log_sf).exp();
    let r2: f64 = x1.iter().zip(x2).zip(log_ls)
        .map(|((a, b), &ll)| ((a - b) / ll.exp()).powi(2))
        .sum();
    let r = r2.sqrt();
    let sqrt5_r = 5.0_f64.sqrt() * r;
    let l_d = log_ls[dim].exp();
    let d_sq = (x1[dim] - x2[dim]).powi(2) / l_d.powi(2);
    // ∂k/∂log(l_d) = σ_f² · 5/3 · d_sq · (1 + √5·r) · exp(−√5·r)
    sigma_f2 * (5.0 / 3.0) * d_sq * (1.0 + sqrt5_r) * (-sqrt5_r).exp()
}
```

### 4. 対数周辺尤度の勾配計算 🔵

**信頼性**: 🔵 *architecture.md「∂L/∂θⱼ = ½ tr((αα^T − K⁻¹) · ∂K/∂θⱼ)」より*

```rust
/// 対数周辺尤度の解析的勾配
/// params = [log_ls_0, ..., log_ls_{d-1}, log_sf, log_sn]
pub(crate) fn log_ml_gradient(
    x: &[Vec<f64>],
    y: &[f64],
    params: &[f64],
) -> Vec<f64> {
    let ndim = x[0].len();
    let log_ls = &params[..ndim];
    let log_sf = params[ndim];
    let log_sn = params[ndim + 1];

    let k = build_kernel_matrix(x, log_ls, log_sf, log_sn);
    let l = match cholesky(&k) {
        Some(l) => l,
        None => return vec![0.0; params.len()],
    };
    let alpha = compute_alpha(&l, y);

    // W = α α^T − K^{-1}  (W[i][j] = alpha[i]*alpha[j] - k_inv[i][j])
    // K^{-1} = L^{-T} L^{-1} → 列ごとに forward/backward sub で計算
    let n = y.len();
    let k_inv = (0..n).map(|j| {
        let e_j: Vec<f64> = (0..n).map(|i| if i == j { 1.0 } else { 0.0 }).collect();
        let v = forward_sub(&l, &e_j);
        backward_sub(&l, &v)
    }).collect::<Vec<_>>();

    let mut grad = vec![0.0; params.len()];

    // ∂L/∂log(l_d) for each dimension d
    for d in 0..ndim {
        let mut tr = 0.0;
        for i in 0..n {
            for j in 0..n {
                let w_ij = alpha[i] * alpha[j] - k_inv[j][i];
                let dk_ij = matern52_ard_grad_ld(&x[i], &x[j], log_ls, log_sf, d);
                tr += w_ij * dk_ij;
            }
        }
        grad[d] = 0.5 * tr;
    }
    // ∂L/∂log(σ_f): ∂k/∂log(σ_f) = 2 * k(x1,x2)
    {
        let mut tr = 0.0;
        for i in 0..n {
            for j in 0..n {
                let w_ij = alpha[i] * alpha[j] - k_inv[j][i];
                let dk_ij = 2.0 * matern52_ard(&x[i], &x[j], log_ls, log_sf);
                tr += w_ij * dk_ij;
            }
        }
        grad[ndim] = 0.5 * tr;
    }
    // ∂L/∂log(σ_n): ∂K/∂log(σ_n) = 2σ_n² I  → tr(W · 2σ_n² I) = 2σ_n² Σ W_ii
    {
        let sigma_n2 = (2.0 * log_sn).exp();
        let tr_w: f64 = (0..n).map(|i| alpha[i].powi(2) - k_inv[i][i]).sum();
        grad[ndim + 1] = sigma_n2 * tr_w;
    }
    grad
}
```

### テストケース1: 対数周辺尤度が実数を返す 🔵

**Then**: `is_finite()` が true（NaN や Inf ではない）

### テストケース2: 勾配の数値微分チェック 🔵

**信頼性**: 🔵 *L-BFGS の正確な動作に解析的勾配が必須*

**Given**: 10 点の 2D データ、パラメータ = [0.0, 0.0, 0.0, -2.0]
**When**: `log_ml_gradient()` の各成分と有限差分（ε=1e-5）を比較
**Then**: 全成分で |解析的 − 数値的| / (|数値的| + 1e-8) < 1e-3

### テストケース3: ノイズ増加で尤度が下がる 🟡

**Given**: クリーンな線形データと同じデータに大きなノイズパラメータ
**When**: 両者の `log_marginal_likelihood()` を比較
**Then**: ノイズが大きい方が尤度が低い（または同等）
