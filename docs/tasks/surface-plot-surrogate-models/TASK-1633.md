### 2. Cholesky 分解（ジッター付き）🔵

**実装ファイル**: `rust_core/src/kriging.rs`

```rust
/// Cholesky 分解: A = L * L^T （下三角行列 L を返す）
/// 数値安定性のためジッター（1e-6）を対角に加算
pub(crate) fn cholesky(a: &[Vec<f64>]) -> Option<Vec<Vec<f64>>> {
    let n = a.len();
    let mut l = vec![vec![0.0_f64; n]; n];
    for i in 0..n {
        for j in 0..=i {
            let mut sum = a[i][j];
            for k in 0..j { sum -= l[i][k] * l[j][k]; }
            if i == j {
                let val = sum + if i == 0 { 1e-6 } else { 0.0 };  // ジッターは初回のみ or 全対角
                if val <= 0.0 { return None; }
                l[i][j] = val.sqrt();
            } else {
                l[i][j] = sum / l[j][j];
            }
        }
    }
    Some(l)
}
// 注: 全対角にジッターを加算する版も検討
// for i in 0..n { let jitter_sum = a[i][i] + 1e-6; ... }
```

### 3. 前進・後退代入 🔵

**信頼性**: 🔵 *GP 学習アルゴリズム（alpha = L^T \ (L \ y)）より*

```rust
/// 前進代入: L * x = b を解く（L は下三角）
pub(crate) fn forward_sub(l: &[Vec<f64>], b: &[f64]) -> Vec<f64> {
    let n = b.len();
    let mut x = vec![0.0; n];
    for i in 0..n {
        let mut s = b[i];
        for j in 0..i { s -= l[i][j] * x[j]; }
        x[i] = s / l[i][i];
    }
    x
}

/// 後退代入: L^T * x = b を解く（L^T は上三角）
pub(crate) fn backward_sub(l: &[Vec<f64>], b: &[f64]) -> Vec<f64> {
    let n = b.len();
    let mut x = vec![0.0; n];
    for i in (0..n).rev() {
        let mut s = b[i];
        for j in (i+1)..n { s -= l[j][i] * x[j]; }  // l[j][i] = L^T[i][j]
        x[i] = s / l[i][i];
    }
    x
}
```

### 4. ARD Matérn 5/2 カーネル 🔵

**信頼性**: 🔵 *architecture.md「ARD Matérn 5/2 を採用・RBF から変更」design-interview.md Q3カーネル変更より*

```rust
/// ARD Matérn 5/2 カーネル
/// k(x1,x2) = σ_f² · (1 + √5·r + 5r²/3) · exp(−√5·r)
/// r² = Σ_d ((x1_d − x2_d) / l_d)²
pub(crate) fn matern52_ard(
    x1: &[f64],
    x2: &[f64],
    log_ls: &[f64],  // 対数長さスケール（ARD: 次元ごと）
    log_sf: f64,     // 対数シグナル分散
) -> f64 {
    let sigma_f2 = (2.0 * log_sf).exp();
    let r2: f64 = x1.iter().zip(x2).zip(log_ls)
        .map(|((a, b), &ll)| {
            let l = ll.exp();
            ((a - b) / l).powi(2)
        })
        .sum();
    let r = r2.sqrt();
    let sqrt5_r = 5.0_f64.sqrt() * r;
    sigma_f2 * (1.0 + sqrt5_r + 5.0 * r2 / 3.0) * (-sqrt5_r).exp()
}
```

### 5. カーネル行列構築 🔵

**信頼性**: 🔵 *dataflow.md「K[i,j] = ARD-Matérn5/2(x_i, x_j) + σ_n² δ_ij」より*

```rust
pub(crate) fn build_kernel_matrix(
    x: &[Vec<f64>],
    log_ls: &[f64],
    log_sf: f64,
    log_sn: f64,
) -> Vec<Vec<f64>> {
    let n = x.len();
    let sigma_n2 = (2.0 * log_sn).exp();
    let mut k = vec![vec![0.0; n]; n];
    for i in 0..n {
        for j in 0..=i {
            let kij = matern52_ard(&x[i], &x[j], log_ls, log_sf);
            k[i][j] = kij;
            k[j][i] = kij;
        }
        k[i][i] += sigma_n2;
    }
    k
}
```

## 単体テスト要件

### テストケース1: Cholesky 分解の正確性 🔵

**Given**: 3×3 の正定値対称行列 A
**When**: `cholesky(A)` を呼び出して L を得る
**Then**: L × L^T の各要素が A と 1e-9 以内で一致する

### テストケース2: 前進・後退代入 🔵

**Given**: 3×3 の Cholesky 因子 L と右辺ベクトル b
**When**: `forward_sub()` → `backward_sub()` で x = K^{-1}b を求める
**Then**: K × x が b と 1e-9 以内で一致する

### テストケース3: Matérn 5/2 の同一点 = σ_f² 🔵

**Given**: x1 = x2 = [1.0, 2.0], log_ls = [0.0, 0.0], log_sf = 0.0
**When**: `matern52_ard()` 呼び出し
**Then**: 結果が 1.0 (= exp(0) = σ_f²) と等しい

### テストケース4: カーネル行列が対称 🔵

**Given**: 5 点の 2D データ
**When**: `build_kernel_matrix()` 呼び出し
**Then**: K[i][j] == K[j][i] が全ての i, j で成立
