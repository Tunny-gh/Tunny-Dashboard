### 2. lib.rs: wasm_compute_pdp_2d シグネチャ更新 🔵

**信頼性**: 🔵 *architecture.md「wasm_compute_pdp_2d に model_type 引数追加」より*

**実装ファイル**: `rust_core/src/lib.rs`

```rust
// 変更前
#[wasm_bindgen(js_name = "computePdp2d")]
pub fn wasm_compute_pdp_2d(
    param1_name: &str,
    param2_name: &str,
    objective_name: &str,
    n_grid: u32,
) -> Result<JsValue, JsValue>

// 変更後
#[wasm_bindgen(js_name = "computePdp2d")]
pub fn wasm_compute_pdp_2d(
    param1_name: &str,
    param2_name: &str,
    objective_name: &str,
    n_grid: u32,
    model_type: &str,  // 追加
) -> Result<JsValue, JsValue>
```

関数本体も `model_type` を `pdp::compute_pdp_2d()` に渡すよう変更:

```rust
match pdp::compute_pdp_2d(param1_name, param2_name, objective_name, n_grid as usize, model_type) {
    Some(result) => { ... }
    None => Err(JsValue::from_str("No active study or parameter not found")),
}
```
