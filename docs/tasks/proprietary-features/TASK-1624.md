### 2. lib.rs への wasm_bindgen エクスポート追加 🔵

**信頼性**: 🔵 *既存wasm_compute_sobol/sensitivity等のlib.rsパターンより*

**実装ファイル**: `rust_core/src/lib.rs`

```rust
#[wasm_bindgen]
pub fn wasm_compute_pdp_2d(
    param1: &str,
    param2: &str,
    objective: &str,
    n_grid: usize,
) -> JsValue {
    let result = pdp::compute_pdp_2d(param1, param2, objective, n_grid);
    let serializer = serde_wasm_bindgen::Serializer::json_compatible();
    result.serialize(&serializer).unwrap_or(JsValue::NULL)
}
```

`use pdp;` および `use serde::Serialize;` のインポートが必要な場合は追加する。
