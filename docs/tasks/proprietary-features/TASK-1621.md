## 実装詳細

### 1. mcdmStore.ts 作成 🔵

**信頼性**: 🔵 *architecture.md McdmStore設計・既存clusterStore.tsパターンより*

**実装ファイル**: `frontend/src/stores/mcdmStore.ts`

```typescript
import { create } from 'zustand'
import { WasmLoader } from '../wasm/wasmLoader'
import { useStudyStore } from './studyStore'
import type { McdmStore, TopsisRankingResult } from '../types'

export const useMcdmStore = create<McdmStore>((set, get) => ({
  topsisWeights: [],       // 初期値は空（Study選択後に目的数で均等配分）
  topsisResult: null,
  isComputing: false,
  topN: 10,
  topsisError: null,

  setTopsisWeights: (weights) => {
    // 合計1.0に正規化
    const sum = weights.reduce((a, b) => a + b, 0)
    const normalized = sum > 0 ? weights.map(w => w / sum) : weights
    set({ topsisWeights: normalized })
  },

  computeTopsis: async () => {
    const wasm = WasmLoader.getInstance()
    const study = useStudyStore.getState().currentStudy
    if (!study || !study.objectiveNames.length) return

    const { topsisWeights } = get()
    const weights = topsisWeights.length === study.objectiveNames.length
      ? topsisWeights
      : study.objectiveNames.map(() => 1 / study.objectiveNames.length)

    set({ isComputing: true, topsisError: null })
    try {
      // gpuBufferからObjective値を取得してflat Float64Arrayを構築
      // ... (gpuBufferまたはgetTrials()からObjective値を収集)
      const result = wasm.computeTopsis(values, nTrials, nObjectives, Float64Array.from(weights), isMinimize)
      set({ topsisResult: { ...result, scores: Array.from(result.scores), rankedIndices: Array.from(result.rankedIndices), positiveIdeal: Array.from(result.positiveIdeal), negativeIdeal: Array.from(result.negativeIdeal) } })
    } catch (e) {
      set({ topsisError: String(e) })
    } finally {
      set({ isComputing: false })
    }
  },

  setTopN: (n) => set({ topN: n }),

  reset: () => set({ topsisResult: null, topsisWeights: [], topsisError: null }),
}))

// Study変更時にリセット（clusterStore.tsと同パターン）
useStudyStore.subscribe(
  (state) => state.currentStudy?.studyId,
  () => useMcdmStore.getState().reset()
)
```

### 2. Objective値の取得方法 🟡

**信頼性**: 🟡 *gpuBuffer構造・getTrials()から妥当な推測*

TOPSISの入力となるObjective値行列は `wasm.getTrials()` から取得する。`values` フィールドが目的関数値の配列。
