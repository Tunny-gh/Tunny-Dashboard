## 実装詳細

### 1. ファイル作成 🔵

**信頼性**: 🔵 *architecture.md TOPSISアルゴリズム概要より*

**実装ファイル**: `rust_core/src/topsis.rs`

TOPSISアルゴリズムの実装手順：

```rust
// 1. 目的関数値行列を正規化（ベクトル正規化）
// r_ij = v_ij / sqrt(sum_i(v_ij^2))
//
// 2. 重み付き正規化行列を計算
// v_ij = w_j * r_ij
//
// 3. 正理想解（各目的のベスト値）と負理想解（ワースト値）を決定
// minimize の場合: A+ = min_i(v_ij), A- = max_i(v_ij)
// maximize の場合: A+ = max_i(v_ij), A- = min_i(v_ij)
//
// 4. 各試行から正・負理想解へのユークリッド距離を計算
// D+_i = sqrt(sum_j(v_ij - A+_j)^2)
// D-_i = sqrt(sum_j(v_ij - A-_j)^2)
//
// 5. スコア = D-_i / (D+_i + D-_i)（0〜1、高い方がベター）
//    D+_i + D-_i = 0 の場合のフォールバック: score = 0.5
//
// 6. スコア降順でindicesをソート

pub struct TopsisResult {
    pub scores: Vec<f64>,         // 全試行のスコア（trial順）
    pub ranked_indices: Vec<u32>, // スコア降順のインデックス
    pub positive_ideal: Vec<f64>, // 正理想解（目的数次元）
    pub negative_ideal: Vec<f64>, // 負理想解（目的数次元）
    pub duration_ms: f64,
}

pub fn compute_topsis(
    values: &[f64],        // [N × M] flattened（行major: trial0_obj0, trial0_obj1, ...）
    n_trials: usize,
    n_objectives: usize,
    weights: &[f64],       // 目的数分の重み（合計1.0に正規化済み前提）
    is_minimize: &[bool],  // 各目的の最小化フラグ
) -> Result<TopsisResult, String>
```
