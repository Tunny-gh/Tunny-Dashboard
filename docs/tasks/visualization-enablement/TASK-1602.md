## 実装詳細

### 1. tunny_core.d.ts への型定義追加 🔵

**信頼性**: 🔵 *REQ-VE-016 / interfaces.ts 設計文書より*

実装ファイル: `frontend/src/wasm/pkg/tunny_core.d.ts`

既存の型定義の末尾に以下を追加：

```typescript
export interface SensitivityWasmResult {
  spearman: number[][];
  ridge: Array<{ beta: number[]; rSquared: number }>;
  paramNames: string[];
  objectiveNames: string[];
  durationMs: number;
}

export interface PcaWasmResult {
  projections: number[][];
  explainedVariance: number[];
  featureNames: string[];
  durationMs: number;
}

export interface KmeansWasmResult {
  labels: number[];
  centroids: number[][];
  wcss: number;
  durationMs: number;
}

export interface ElbowWasmResult {
  wcssPerK: number[];
  recommendedK: number;
  durationMs: number;
}

export interface ClusterStatsWasmResult {
  stats: Array<{
    clusterId: number;
    size: number;
    centroid: Record<string, number>;
    std: Record<string, number>;
    significantDiffs: string[];
  }>;
  durationMs: number;
}
```

### 2. tunny_core.d.ts への関数シグネチャ追加 🔵

**信頼性**: 🔵 *REQ-VE-010〜015 / wasm-api.md パターンより*

既存関数（`export function parseJournal(...)` 等）と同列に以下を追加：

```typescript
export function computeSensitivity(): SensitivityWasmResult;
export function computeSensitivitySelected(indices: Uint32Array): SensitivityWasmResult;
export function runPca(n_components: number, space: string): PcaWasmResult;

export function estimateKElbow(data: Float64Array, n_cols: number, max_k: number): ElbowWasmResult;
```

### 3. wasmLoader.ts へのメソッドプロパティ追加 🔵

**信頼性**: 🔵 *REQ-VE-020 / wasmLoader.ts 既存パターンより*

実装ファイル: `frontend/src/wasm/wasmLoader.ts`

既存の `computeHvHistory`・`getTrials` 等と同じパターンでクラスにプロパティを追加：

```typescript
// import 追加
import type {
  SensitivityWasmResult, PcaWasmResult, KmeansWasmResult,
  ElbowWasmResult, ClusterStatsWasmResult
} from './pkg/tunny_core';

// クラスプロパティ追加（既存プロパティの後に追加）
computeSensitivity!: () => SensitivityWasmResult;
computeSensitivitySelected!: (indices: Uint32Array) => SensitivityWasmResult;
runPca!: (nComponents: number, space: string) => PcaWasmResult;
runKmeans!: (k: number, data: Float64Array, nCols: number) => KmeansWasmResult;
estimateKElbow!: (data: Float64Array, nCols: number, maxK: number) => ElbowWasmResult;
computeClusterStats!: (labels: Int32Array) => ClusterStatsWasmResult;
```

### 4. wasmLoader.ts _initialize() へのバインディング追加 🔵

**信頼性**: 🔵 *REQ-VE-021 / wasmLoader.ts 既存 _initialize() パターンより*

`_initialize()` 内に以下を追加（既存の `this.computeHvHistory = ...` と同じパターン）：

```typescript
this.computeSensitivity = () => wasm.computeSensitivity() as SensitivityWasmResult;
this.computeSensitivitySelected = (indices) =>
  wasm.computeSensitivitySelected(indices) as SensitivityWasmResult;
this.runPca = (nComponents, space) => wasm.runPca(nComponents, space) as PcaWasmResult;
this.runKmeans = (k, data, nCols) => wasm.runKmeans(k, data, nCols) as KmeansWasmResult;
this.estimateKElbow = (data, nCols, maxK) =>
  wasm.estimateKElbow(data, nCols, maxK) as ElbowWasmResult;
this.computeClusterStats = (labels) =>
  wasm.computeClusterStats(labels) as ClusterStatsWasmResult;
```
