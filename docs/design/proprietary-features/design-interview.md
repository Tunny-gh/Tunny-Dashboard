## ヒアリング結果サマリー

### 確認できた事項
- 追加機能: 3D応答曲面プロット（`surface3d`）+ TOPSISランキング（`topsis-ranking`）の2機能
- 描画技術: deck.gl GridLayer（3D曲面）+ ECharts BarChart（TOPSISランキング）
- TOPSIS: Rust/WASMで実装、50,000点でも100ms以内を目標
- UI統合: 既存ChartId/ChartCatalog方式に沿って追加
- 既存転用: `compute_pdp_2d()` を3D曲面のデータソースとして転用可能

### 設計方針の決定事項
- `mcdmStore`（新設）でTOPSIS状態管理
- `analysisStore` に `surrogateModelType`・`surface3dCache` を追加（拡張）
- 5層アーキテクチャ踏襲（WASM → d.ts → wasmLoader → Store → Component）
- Tailwind CSS不使用（インラインスタイルのみ）

### 残課題
- `compute_pdp_2d()` は Rust 実装済みだが `wasmLoader.ts` に未公開（→ `lib.rs` に `wasm_bindgen` エクスポート追加が確定必要）
- TOPSIS計算でDivision by Zero（D+ + D- = 0）の場合のフォールバック処理は実装時に検討
- deck.gl `GridLayer` が z値ヒートマップとして使えるか、`SurfaceLayer` の有無は実装時に確認
